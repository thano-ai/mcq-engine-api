1. What is the primary goal of a Zero Trust Architecture?
A) Perimeter security
B) Continuous verification
C) Network caching
D) VPN tunneling
Answer: B

2. Which protocol operates at Layer 3 of the OSI model?
A) HTTP
B) TCP
C) IP
D) FTP
Answer: C

3. What does DNS stand for?
A) Domain Name System
B) Dynamic Network Service
C) Data Node Server
D) Digital Name Storage
Answer: A

4. Which HTTP method is idempotent?
A) POST
B) PATCH
C) GET
D) CONNECT
Answer: C

5. What is the default port for HTTPS?
A) 80
B) 443
C) 8080
D) 22
Answer: B
